MARQUEE

MARQUEE is a bare bones widget that renders basic HTML using TTF fonts and immediate mode rendering. Only a few HTML tags are supported that are minimally sufficient to render basic documentation in games and simple help systems.

Basic supported features:

  • Headings (h1 through h6) with proper font sizing
  • Paragraphs with automatic word wrapping
  • Bold text rendering with proper fonts
  • Italic text rendering with proper fonts
  • Hyperlinks: Visit raylib.com
  • Horizontal rules for visual separation
  • Both ordered and unordered lists

Technical Implementation

The widget uses immediate mode rendering where each frame:

  1. Parses HTML into structured elements
  2. Calculates layout and positioning
  3. Renders elements using raylib drawing functions
  4. Handles user interaction (scrolling, clicking links)
