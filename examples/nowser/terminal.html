Terminal Commands Reference

Essential commands for daily development work

File Operations

Navigate directories with cd and list contents with ls -la.

Common Commands

  • List files: ls -la or dir on Windows
  • Change directory: cd /path/to/directory
  • Create directory: mkdir new-folder
  • Remove files: rm filename or rm -rf folder/

Git Workflow

Here's a typical Git workflow for contributing to a project:

# Clone the repository
git clone https://github.com/user/repo.git
cd repo

# Create a new branch
git checkout -b feature/new-feature

# Make changes and commit
git add .
git commit -m "Add new feature"

# Push to remote
git push origin feature/new-feature

# Create pull request (on GitHub/GitLab)

Useful Git Commands

  1. Check status: git status
  2. View history: git log --oneline
  3. Undo changes: git reset --hard HEAD
  4. Switch branches: git checkout main

System Information

Get system details using these commands:

# Check disk usage
df -h

# Memory usage
free -h

# Process list
ps aux | grep process-name

# Network connections
netstat -tulpn

Package Management

Install packages on different systems:

  • Ubuntu/Debian: sudo apt install package-name
  • CentOS/RHEL: sudo yum install package-name
  • macOS: brew install package-name
  • Arch Linux: sudo pacman -S package-name

For more help, try man command-name or command-name --help.
